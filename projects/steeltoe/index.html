Steeltoe
cloud • spring • steeltoe

Not soon after joining Pivotal I was introduced to the Steeltoe project. Orginally it was meant to bring Spring Cloud functions to .NET applications. Today it still keeps that core value but has expanded dramatically into other cloud functions to help .NET developers. It has also kept current with .NET’s evolution but supprting 2 different branches for legacy(Framework) and modern(Core & Standard) .NET.
